# Plano de testes para UI

## Objetivo

Validar os principais fluxos da respectiva aplicação (Sauce Demo), garantindo assim o correto funcionamento das funcionalidades críticas de autenticação, gerenciamento de produtos, carrinho e checkout.

---

## Escopo do projeto

## Funcionalidades testadas

- Login
- Logout
- Ordenação dos produtos
- Gerenciamento do carrinho de compras
- Checkout
- Navegação entre as páginas

---

## Tipos de teste

- Funcionais
- Automatizados end-to-end
- Validação de fluxos críticos

---

## Critérios de aceite

- Usuário consegue autenticar com credenciais válidas
- Usuário recebe feedback em autenticação inválida
- Produtos são corretamente ordenados
- Fluxo de compra é concluído com sucesso
- Carrinho reflete corretamente os produtos adicionados/removidos

--- 

## Ferramentas utilizadas

- Playwright
- TypeScript
- Chromium Browser

---

## Evidências

As evidências de execução dos testes estão disponíveis na pasta `/evidences`.
